> *Technology should not feel like technology. It should feel like a stone smoothed by a river, or a page turned in a book.*

## Brand Philosophy

In an age of infinite distraction, **Nami** designs objects that respect your silence. We reject the hyper-stimulated flashing metrics of conventional consumer electronics in favor of a high tactile philosophy. We use materials that age with grace—sandstone composite, untreated aluminum, and organic linen—creating a peaceful, tactile bridge between the digital world and your physical home.

## Getting Started

### Prerequisites

* Node.js (version 18 or higher)
* A Gemini API key (from Google AI Studio)

### Installation & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/black12-ag/nami-quiet-living.git
   cd nami-quiet-living
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory:
   ```env
   GEMINI_API_KEY=your_actual_gemini_api_key_here
   ```

4. Launch the local development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
